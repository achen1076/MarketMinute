# MarketMinute Business

Standalone landing page for the MarketMinute Business product - AI-powered financial modeling.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server (port 3001)
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deployment

This is a separate Vercel project that should be deployed to `business.marketminute.io`.

### Vercel Setup

1. Create a new Vercel project from the `business` folder
2. Set the root directory to `business`
3. Add `business.marketminute.io` as a domain

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript
